How to use Prism-UI

        
      

      

        

          
Getting Started

          
    
            
  • Projects — Create a project for each task or topic. Link a local folder to keep documents nearby.
    • 
            
  • Chat — Type a message and press Enter to send. The AI will respond based on your settings.
    • 
            
  • Routes — The colored badges above the chat show the AI's response strategy (comply, challenge, clarify, etc.).
    • 
          

        

        

          
Panels

          
    
            
  • Analysis (top-right) — Inspect bias detection, confidence scores, and session metrics.
    • 
            
  • Memory (top-right) — Save facts and context across sessions. Hot/Warm/Cool tiers auto-decay.
    • 
            
  • Workspace — Link a folder to any project. Click file chips to inject content into chat.
    • 
          

        

        

          
Tips

          
    
            
  • Click a bias badge under your message to open detailed analysis.
    • 
            
  • Hover over any metric in Analysis for a description of what it means.
    • 
            
  • Use the Settings modal to change model, creativity, and system prompt.
    • 
            
  • Plugins add new skills — try enabling the Summarize or Web Search plugins.
    •
